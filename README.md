## Setup Steps

1. Setup llama.cpp
2. download model[https://huggingface.com/unsloth/Qwen3.5-4b-GGUF]
3. use llama-server to host Qwen3.5-4b (Q4_K_S or XL)
4. Install mcp-filesystem: `pip install mcp-filesystem`
5. Build tools suite, custom sys prompt and thinking jinja prompt
